Welcome!

This application is a an enterprise content management system built using HDP. The specific components in this architecture are the following:

  • HBase as the system of Record for Documents
  • Solr as the System of Record for Metadata and providing blazing fast search
  • Spring Data Solr to provide easy abstraction on top of Solr
  • Spring HBase to provide easy abstraction on top of Hbase
